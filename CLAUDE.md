# Writing Portfolio — Rules for AI

When adding or editing blog entries in `writing/data.js`, always follow these rules:

## Tags
- Every entry must have **exactly 8 tags**
- Keep: topic category, location, program names (Mass Save, LogisVert, CleanBC, etc.), technology type
- Drop: brand names, redundant synonyms, overly specific sub-terms

## Excerpt
- Every excerpt must be **700–900 characters** (count every character including spaces)
- Same professional tone as existing entries
- No invented facts — only details supported by the title and topic

## Verification
Before pushing any changes to data.js, count the tags and excerpt length for every entry you touched and confirm both rules are met.
